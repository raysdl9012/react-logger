---
description: How to run the ionic-react-logger demo application
---

To run the demo application and see the Liquid Glass logger in action, follow these steps:

1. **Build the Library** (from the root directory):
```bash
npm run build
```

2. **Run the Demo** (via the example folder):
// turbo
```bash
cd example && npm install && npm run dev
```

3. **Open the Browser**:
Navigate to the URL shown in the terminal (usually `http://localhost:5173`).

4. **Interact**:
Use the buttons to generate logs and click the bug icon (üêû) to open the console.
